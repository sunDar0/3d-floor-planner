# Simple Floor Planner (Floor Planner Pro)

웹 브라우저에서 바로 실행 가능한 3D 평면도 에디터입니다. 별도의 설치 없이 HTML 파일 하나로 2D 도면 작성부터 3D 1인칭 관람까지 가능합니다.
## 데모 주소: [페이지 이동](https://sundar0.github.io/3d-floor-planner/)
## ✨ 주요 기능

### 1. 2D 도면 편집 (2D Editor)
*   **벽 & 방 그리기**: 직관적인 드래그 앤 드롭으로 벽과 바닥을 생성합니다.
*   **개구부 배치**: 문, 창문, 발코니 창을 벽에 자동으로 스냅하여 배치합니다.
*   **가구 배치**: 침대, 소파, 책상, 냉장고 등 다양한 가구를 배치할 수 있습니다.
*   **가구 편집**:
    *   **이동**: 드래그하여 위치 변경
    *   **회전**: `R` 키를 눌러 90도씩 회전
    *   **삭제**: `Delete` 또는 `Backspace` 키로 삭제
*   **편의 기능**:
    *   **스마트 스냅**: 그리드 및 다른 객체에 맞춰 자동 정렬
    *   **실행 취소/다시 실행**: `Ctrl+Z`, `Ctrl+Y` 지원
    *   **줌 & 팬**: 마우스 휠로 확대/축소, 우클릭/휠클릭 드래그로 화면 이동
*   **저장/불러오기**: 작업 내용을 JSON 파일로 저장하고 다시 불러올 수 있습니다.

### 2. 3D 뷰어 (3D Viewer)
*   **모드 전환**: 버튼 하나로 2D 도면을 즉시 3D로 변환하여 확인합니다.
*   **조감도 모드 (Orbit View)**: 자유롭게 회전하며 전체적인 구조를 파악할 수 있습니다.
*   **1인칭 관람 모드 (First Person View)**:
    *   마치 집 안을 걸어 다니는 듯한 경험을 제공합니다.
    *   **WASD** 키로 이동, 마우스로 시선 처리.
    *   벽 충돌 방지(Collision Detection) 적용.
    *   **성별 키 설정**: 남성(180cm), 여성(160cm) 눈높이 조절 가능.

## 🛠 기술 스택
이 프로젝트는 별도의 빌드 과정 없이 브라우저에서 바로 실행되도록 구성되었습니다.
*   **Core**: HTML5, JavaScript (ES6+)
*   **Framework**: React 18 (CDN)
*   **Styling**: Tailwind CSS (CDN)
*   **3D Graphics**: Three.js (CDN)
*   **Transpiler**: Babel Standalone (브라우저 내 JSX 컴파일)

## 🚀 실행 방법
1. 저장소의 `3d-floor-planner.html` 파일을 다운로드합니다.
2. 파일을 크롬(Chrome), 엣지(Edge) 등 최신 웹 브라우저로 엽니다.
3. 바로 사용하실 수 있습니다!

## 🎮 단축키 가이드

| 기능 | 단축키 |
|---|---|
| **실행 취소** | `Ctrl + Z` |
| **다시 실행** | `Ctrl + Y` 또는 `Ctrl + Shift + Z` |
| **가구 회전** | `R` |
| **선택 삭제** | `Delete` 또는 `Backspace` |
| **선택 취소** | `Esc` |
| **3D 이동** | `W`, `A`, `S`, `D` |
| **3D 시선** | 마우스 이동 |
| **3D 커서 잠금 해제** | `Esc` |
